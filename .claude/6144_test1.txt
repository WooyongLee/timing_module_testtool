❯   MQTT 0x44 명령 프로토콜:                                                    
  │     명령           │        형식               │               동작         
  |                                                                             
  │ AD9361 초기화 │ 0x44 0x00              │ AD9361 61.44MHz 모드 초기화        
  │                                                                             
  │ FFT                 │ 0x44 0x01 <fft_len>│ FFT 수행 후 스펙트럼             
  데이터 전송   │                                                               
  │ IQ Capture      │ 0x44 0x02 <length>│ IQ 데이터 캡처 후 전송                
  │                                                                             
                                                                                
  응답:                                                                         
  - 성공: 0x44 0x00 OK, 0x44 0x01 OK, 0x44 0x02 OK                              
  - 실패: 0x44 0x00 FAIL 등                                                     
                                                                                
  위의 명령 프로토콜로 기존에 I/Q Capture 및 Spectrum 에 대해서 수정이          
  가능할까요.                                                                   
  기존에는 Start 누름 시 연속적으로 데이터를 받는 식이었다면, FFT 또는 I/Q      
  요청을 1회성 클릭으로 Single 방식으로 데이터를 받아오는 방식으로              
  바뀝니다.                                                                     
  UI도 바뀌어야 한다면 수정해주세요.
