  61.44MHz Test Mode MQTT 프로토콜 문서

  개요

  - Command Header: 0x44 (CMD_61_44_TEST)
  - Response Header: 0x45 (CMD_61_44_TEST_RESP)
  - 샘플링 레이트: 61.44MHz
  - 출력 스펙트럼 포인트: 8192개

  ---
  UI → FW (요청 명령)
  명령: 초기화
  토픽: pact/command
  헤더: 0x44 0x00
  데이터: <freq_hz> <rbw_idx> <max_hold>
  Description: 61.44MHz 테스트 모드 초기화 및 파라미터 설정
  ────────────────────────────────────────
  명령: 스펙트럼 측정
  토픽: pact/command
  헤더: 0x44 0x01
  데이터: (없음)
  Description: FFT 기반 스펙트럼 측정 요청
  ────────────────────────────────────────
  명령: IQ 캡처
  토픽: pact/command
  헤더: 0x44 0x02
  데이터: <num_samples>
  Description: IQ 데이터 캡처 요청
  ────────────────────────────────────────
  명령: 정지
  토픽: pact/command
  헤더: 0x44 0x0F
  데이터: (없음)
  Description: 측정 정지


  파라미터 상세
  파라미터: freq_hz
  타입: uint64
  범위: -
  설명: 중심 주파수 (Hz), 예: 3000000000 (3GHz)
  ────────────────────────────────────────
  파라미터: rbw_idx
  타입: uint32
  범위: 0-3
  설명: RBW 인덱스: 0=15kHz, 1=30kHz, 2=60kHz, 3=120kHz
  ────────────────────────────────────────
  파라미터: max_hold
  타입: uint32
  범위: 0-1
  설명: MaxHold 모드: 0=OFF, 1=ON
  ────────────────────────────────────────
  파라미터: num_samples
  타입: uint32
  범위: 1-65536
  설명: IQ 샘플 수 (기본값: 8192)
  ---
  FW → UI (응답)
  ┌────────────┬────────────┬─────────┬───────────────────┬────────────────────┐
  │ 응답 타입  │    토픽    │  헤더   │      데이터       │    Description     │
  ├────────────┼────────────┼─────────┼───────────────────┼────────────────────┤
  │ 초기화     │            │         │ <result> <freq>   │ 초기화 결과        │
  │ 성공       │ pact/data1│ 0x45    │ <rbw> <max_hold>  │ (result: 1=성공,   │
  │            │            │         │                   │ 0=실패)            │
  ├────────────┼────────────┼─────────┼───────────────────┼────────────────────┤
  │ 스펙트럼   │            │         │                   │ 스펙트럼 측정      │
  │ 데이터     │ pact/data1│ 0x45    │ 0x01 <num_points> │ 성공, 데이터 개수  │
  │ 헤더       │            │         │                   │ (8192)             │
  ├────────────┼────────────┼─────────┼───────────────────┼────────────────────┤
  │ 스펙트럼   │            │ (byte   │                   │ 스펙트럼 데이터    │
  │ 데이터     │ pact/data2│ array)  │ ds32s[8192]       │ (dBm × 100, 32768  │
  │ 본체       │            │         │                   │ bytes)             │
  ├────────────┼────────────┼─────────┼───────────────────┼────────────────────┤
  │ IQ 데이터  │ pact/data1 │ 0x45    │ 0x02              │ IQ 캡처 성공, 샘플 │
  │ 헤더       │            │         │ <num_samples>     │  수                │
  ├────────────┼────────────┼─────────┼───────────────────┼────────────────────┤
  │ IQ 데이터  │            │ (byte   │                   │ IQ 데이터          │
  │ 본체       │ pact/data2 │ array)  │ ds16s[n×2]        │ (interleaved I/Q,  │
  │            │            │         │                   │ 16-bit)            │
  ├────────────┼────────────┼─────────┼───────────────────┼────────────────────┤
  │ 정지 응답  │ pact/data1 │ 0x45    │ 0x0F              │ 측정 정지 완료     │
  ├────────────┼────────────┼─────────┼───────────────────┼────



  데이터 포맷 상세

  스펙트럼 데이터 (TYPE_61_44_SPECTRUM = 0x01)

  응답 1: "0x45 0x01 8192"        // 헤더 + 포인트 수
  응답 2: byte array              // ds32s[8192] = 32768 bytes
          - 단위: dBm × 100 (예: -5000 = -50.00 dBm)
          - FFT 중심 8192 포인트

  IQ 데이터 (TYPE_61_44_IQ_CAPTURE = 0x02)

  응답 1: "0x45 0x02 <num_samples>"   // 헤더 + 샘플 수
  응답 2: byte array                   // ds16s[num_samples × 2]
          - Interleaved: I0, Q0, I1, Q1, ...
          - 각 값: 16-bit signed integer
          - 총 바이트: num_samples × 4 bytes

  ---
  사용 예시

  1. 초기화 (3GHz, RBW=30kHz, MaxHold OFF)
  UI → FW: "0x44 0x00 3000000000 1 0"
  FW → UI: "0x45 1 3000000000 1 0"

  2. 스펙트럼 측정
  UI → FW: "0x44 0x01"
  FW → UI: "0x45 0x01 8192"
  FW → UI: [32768 bytes spectrum data]

  3. IQ 캡처 (4096 샘플)
  UI → FW: "0x44 0x02 4096"
  FW → UI: "0x45 0x02 4096"
  FW → UI: [16384 bytes IQ data]

  4. 측정 정지
  UI → FW: "0x44 0x0F"
  FW → UI: "0x45 0x0F"

  상수 정의 (command.h)

  #define CMD_61_44_TEST      0x44
  #define CMD_61_44_TEST_RESP "0x45"

  #define TYPE_61_44_INIT         0x00  // Initialize
  #define TYPE_61_44_SPECTRUM     0x01  // Spectrum measurement
  #define TYPE_61_44_IQ_CAPTURE   0x02  // IQ data capture
  #define TYPE_61_44_STOP         0x0F  // Stop measurement
